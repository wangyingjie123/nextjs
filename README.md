#启动项目<br>
1、下载压缩文件或运行
```
git clone https://github.com/wangyingjie123/svip-vue.git
```
2、安装依赖
```
npm install
```
3、启动项目
```
npm run dev
 ```
4、如果要打包，运行
```
npm run build
```
